# Happy Birthday — Sudha

This is a small, romantic birthday webpage for Sudha created by Lokesh.

What you get:
- `index.html` — the webpage
- `styles.css` — styles and responsive layout
- `script.js` — surprise card, floating hearts, audio control

How to use:
1. Place a romantic instrumental MP3 at `assets/romantic.mp3` (or change the `audio` src in `index.html`).
2. Replace the photo placeholder by adding an image inside the `.photo-placeholder` element.
3. Open `index.html` in a browser (double-click or serve with a simple HTTP server).

Notes:
- The audio file is not included. Use a short, royalty-free instrumental.
- Colors and fonts use Google Fonts; internet required for the font link.
